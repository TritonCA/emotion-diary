# Дневник эмоций — гайд

Минималистичный трекер настроения. Записывайте эмоции, ставьте интенсивность, ведите дневник, смотрите статистику и не забывайте отмечаться по напоминаниям.

> Скриншоты лежат в [`_assets/`](./_assets/). Замените файлы своими — имена в этом гайде должны совпадать с реальными PNG.

---

## Запись эмоции

![Экран записи](./_assets/record.png)

- Дата/время в верхней карточке — тап открывает date + time picker. По умолчанию «Сейчас».
- **Текущие эмоции** → тап `+` → открывает каталог.
- **Что произошло?** — свободный текст (триггер / контекст). Виден потом в истории.
- **«Сохранить запись»** — фиксирует запись в дневнике.

### Выбор эмоций

![Каталог эмоций](./_assets/record_picker.png)

- 9 категорий: радость, спокойствие, любовь, интерес, грусть, гнев, страх, удивление, усталость.
- Тап по категории → конкретные эмоции внутри. Можно отметить несколько, есть «Выбрать все».
- Поле сверху ищет и по названию категории, и по эмоции (на текущем языке + английском).

### Интенсивность

![Одиночная эмоция](./_assets/record_intensity_single.png)

- При одной выбранной эмоции — большой ползунок 0–10. Цвет визуализации непрерывно меняется от приглушённого к насыщенному и зависит от валентности (позитив = зелёный, негатив = красный, нейтрал = акцентный цвет).

![Несколько эмоций](./_assets/record_intensity_multi.png)

- При 2+ эмоциях — отдельный слайдер для каждой. Общая интенсивность считается как среднее и показывается крупно.

## История

![Список записей](./_assets/history.png)

- Группировка по дням (Сегодня / Вчера / явная дата).
- Чипы сверху: фильтр по категории + период (7 / 30 / 365 / всё время).
- В каждой карточке: время, цветные «пилюли» эмоций с их собственной интенсивностью, общая интенсивность справа, цитата триггера.

### Редактирование / удаление записи

![Редактирование записи](./_assets/entry_edit.png)

- **Тап** по карточке → открывает запись на редактирование. Все поля те же, что и при создании.
- Кнопка корзины в шапке → удаление с подтверждением.
- «Сохранить изменения» обновляет запись без создания дубля.

## Статистика

![Статистика](./_assets/stats.png)

- Селектор периода: неделя / месяц / год / всё.
- Фильтр по категории — статистика пересчитывается под выбранную (или «Все категории»).
- Метрики: всего записей, средняя интенсивность, топ-категория.
- График интенсивности — реальный тренд за последние 7 дней.
- Частота — топ-4 категории в процентах.
- Контекст — самые частые слова из триггеров.
- Donut — соотношение позитив / нейтрал / негатив, в центре доминирующая «фокус»-валентность.

## Напоминания

![Список напоминаний](./_assets/reminders.png)

Открываются из Настроек → «Управление напоминаниями».

- **+ Добавить** внизу справа.
- Свитч у каждой карточки — быстрое вкл/выкл.
- Тап по карточке → редактирование.

### Создание / редактирование напоминания

![Редактирование напоминания](./_assets/reminder_edit.png)

- **Текст уведомления** — что появится в шторке. Есть дефолт по умолчанию, если поле пустое.
- **Время** — час и минута.
- **Периодичность**: один раз / каждые N часов / каждые N дней / каждые N недель.
- **Интервал** — степпер N для периодических напоминаний (от 1 до 365).
- **Включено** — переключатель.

> Уведомления опираются на системный планировщик Android. На некоторых производителях (Xiaomi, Huawei) ОС может убивать фоновые задачи — на самой странице есть жёлтая плашка с предупреждением и советом добавить приложение в автозапуск.

## Настройки

![Настройки](./_assets/settings.png)

- **Внешний вид**
  - Тема: светлая / тёмная / системная.
  - Язык: русский / English.
  - Акцентный цвет: индиго, бирюза, роза, янтарь, изумруд, графит.
- **Напоминания** → переход к списку (см. выше).
- **Данные**
  - Управление эмоциями.
  - Экспорт в CSV (через системный share).
  - Импорт из CSV.
  - Удалить все данные (с подтверждением).
- **О приложении** → исходный код на GitHub.

Все настройки сохраняются автоматически — после перезапуска приложения вы видите тот же язык, тему и акцент.

### Управление эмоциями

![Управление эмоциями](./_assets/manage_emotions.png)

- Раскройте категорию → список эмоций.
- **Тап** по эмоции → меню «Переименовать / Удалить».
- Кнопка «+ Добавить» в каждой категории — для своих эмоций.
- Удалённые базовые эмоции пропадают из выбора, но **остаются в прошлых записях** в исходном виде — история не переписывается задним числом.

## CSV-формат (импорт/экспорт)

Колонки: `id, timestamp, emotions, intensity, trigger`

- `timestamp` — ISO-8601.
- `emotions` — список через `;`, каждая запись = `name|categoryId|valence|intensity`. Старый трёхчастный формат без интенсивности тоже понимается.
- `intensity` — общая интенсивность строки 0..10 (среднее по эмоциям).
